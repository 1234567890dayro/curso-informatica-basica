# Comandos por consola

## Consola windows

| Comando | Descripción |
| --- | --- |
| cls | Limpia la pantalla |
| cd / cd .. | Cambia de directorio |
| mkdir | Crea un nuevo directorio |

## Consola Linux

| Comando | Descripción |
| --- | --- |
| clear | Limpia la pantalla |
| cd / cd .. | Cambia de directorio |
| mkdir | Crea un nuevo directorio |
